# PiCar
A Raspbeery Pi Pico based Autonomous Vehicle Solution

## Hardware
- MCU: Raspberry Pi Pico 2/2W
- Motor Driver: L298N
- Power: 12680 Battery
- Motor: DC Motor
- Sensors: Ultrasonic Sensors
